Overview

This project demonstrates a proof of concept (PoC) for an AI-powered IT support assistant. It leverages MCP Server, and Large Language Models (LLMs) to automatically answer employee IT queries by retrieving information from internal databases and providing actionable solutions.

Features

- Answers common IT support queries using LLMs
- Retrieves information from internal databases for accurate responses

Technologies Used

- Python
- MCP Server (Multi-Client Platform)
- LangGraph
- Large Language Models (LLMs) - Gemini AI
- Databases (MySQL / internal DB connection)
